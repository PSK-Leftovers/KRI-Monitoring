# Sukurti vartotojo prisijungimus prie įmonės resursų 


KAIP adminas
NORIU sukurti vartotojo prisijungimus prie įmonės resursų
KAD skirtingi vartotojai galėtų naudotis sistema



Santrauka

Vartotojo sąsaja per kurią administratorius gali sukurti prisijungimą pagal darbuotojo el paštą



Tinkamumo kriterijai

AC1: Gali įvesti el. paštą ir slaptažodį
AC2: Sistema patikrina unikalumą
AC3: Sukurta veikianto vartojo sąsaja, pridėtos ekrano nuotraukos

AC4: Slaptažodis turi būti min. 8 simboliai

AC5: Priskirti rolę: ADMIN arba USER

AC6: Jei el. paštas jau egzistuoja tada rodoma klaida
